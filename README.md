Brief Explanation of Design Decisions

HTML:

Simple structure with Header, Main, and Footer.

Added a button for interaction and a paragraph to show current color.

CSS:

Used responsive design so webpage looks good on desktop & mobile.

Added smooth transitions for background color changes.

Button has hover effect for better user experience.

JavaScript:

Added interactivity: button changes background color randomly.

Displays current color name so user knows what color is applied.

Testing & usability:

Tested on different browser sizes.

Ensured webpage is visually clean, simple, and user-friendly.
